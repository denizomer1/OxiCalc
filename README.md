# OxiCalc 🚀📱⌚

A professional-grade, multi-platform calculator ecosystem built with Jetpack Compose. Optimized for Phones, Tablets, and Wear OS watches.

## Features
- **Multi-Platform:** Seamless experience across Android smartphones, tablets, and smartwatches.
- **Wear OS Native:** Mathematically optimized circular UI with a unique side-tab navigation.
- **Material You:** Dynamic color support that adapts to your system theme.
- **Global Localization:** Supported in 8 languages (EN, TR, DE, FR, ES, RU, JA, AR).
- **Privacy First:** No data collection. All history is stored locally on the device.

## Architecture
This project follows a clean, modular architecture:
- `:shared`: Core calculation logic and shared view models.
- `:app`: Phone and Tablet specific UI.
- `:wear`: Specialized Wear OS interface.
